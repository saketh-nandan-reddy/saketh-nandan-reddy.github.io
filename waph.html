WAPH Course Summary

Course: Web Application Programming and Hacking

Instructor: Dr. Phu H. Phung

Why Learn Web Application Security?

Understanding web security is essential for protecting applications from vulnerabilities that could expose user data or disrupt service. This course empowers students to think like both attackers and defenders, helping them build secure, resilient applications from the ground up.

What You'll Gain

  • Mastery of HTTP protocols and network-level traffic inspection
  • Hands-on experience with real-world attack simulations and mitigations
  • Full-stack debugging using JavaScript, Ajax, and PHP
  • Deployment of web projects on GitHub with live testing
  • Understanding of authentication, cookies, sessions, and secure coding

Course Topics

  • HTTP Analysis using Telnet and Wireshark
  • CGI with C, PHP Basics
  • JavaScript and Ajax Integration
  • Front-End Deployment on GitHub Pages
  • XSS, SQLi, CSRF Attack Simulation
  • Session Authentication & Secure Coding

Hands-On Projects

  • Lab 1: HTTP Protocol Analysis
  • Lab 2: Front-end Development with Ajax & APIs
  • Hackathon 1: XSS Attack Simulation
  • Personal Project 1: Profile Website with API Integration (this site)
